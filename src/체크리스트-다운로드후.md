## 💻 3단계: Node.js 확인

Cursor 하단에서 터미널 열기 (`Ctrl + ~`)

```bash
node --version
```

**기대 결과:**
```
v18.17.0  (또는 v16 이상의 버전)
```

❌ 에러가 나면?
→ https://nodejs.org 에서 Node.js 다운로드 설치

→ Tailwind CSS 문제. 아래 명령어 실행:

```bash
# 터미널에서 Ctrl+C로 서버 종료 후
rm -rf node_modules package-lock.json
npm install
npm run dev
```

## 🆘 문제 해결

### 가장 흔한 문제 3가지

**1. "npm: command not found"**
→ Node.js 설치 필요

**2. 스타일 안 나옴 (흰 화면)**
→ `npm install` 다시 실행

**3. 포트 3000 사용 중**
→ `vite.config.ts`에서 포트 변경 (3001로)
