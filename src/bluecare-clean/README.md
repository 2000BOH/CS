# 🚀 BLUECARE - 순수 HTML 버전

**이 폴더의 파일만 사용하세요!**

## 📁 필요한 파일 (4개만!)

- `index.html` - 메인 HTML
- `style.css` - 스타일
- `script.js` - JavaScript
- `config.js` - Supabase 설정

## ✅ 실행 방법

1. **bluecare-clean 폴더** 전체를 복사
2. 새 폴더에 붙여넣기
3. `index.html` 더블클릭!

## 🔧 또는 웹 서버로 실행

```bash
# Python 3
python -m http.server 8000

# 브라우저에서
http://localhost:8000
```
